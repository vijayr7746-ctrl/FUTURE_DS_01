# FUTURE_DS_01
Analyze business sales data to identify revenue trends, top-selling
products, high-value categories, and regional performance.

Project Overview
This project analyzes business sales performance using Python and MS Excel. The analysis helps identify sales trends, profitable categories, top-performing products, and regional sales performance. The project provides meaningful business insights using data visualization and dashboard analytics.

**Objectives**
- Analyze overall sales performance
- Identify top-selling products
- Compare regional sales performance
- Analyze category-wise sales
- Track monthly sales trends
- Generate business insights and recommendations

**Tools Used**
- Python
- Pandas
- Matplotlib
- MS Excel

# Dataset Used
Superstore Dataset

## Features
- Total Sales Analysis
- Profit Analysis
- Top Products Analysis
- Region-wise Sales Analysis
- Monthly Sales Trend Analysis
- Data Visualization
- Excel Dashboard Creation
